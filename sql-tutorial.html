SQL Introduction

SQL (Structured Query Language) is the standard language for relational databases. Use SELECT to retrieve data, INSERT to add, UPDATE to modify, and DELETE to remove. Below is a live SQLite database - run queries instantly!

SQL Example
SELECT * FROM Users;

SELECT name, email FROM Users WHERE age > 25;

SQL WHERE & Conditions

The WHERE clause filters records. Use operators like =, <>, >, <, >=, <=, BETWEEN, LIKE, and IN. Combine conditions with AND and OR.

SQL Example
SELECT * FROM Users WHERE age > 25;

SELECT * FROM Products WHERE price BETWEEN 20 AND 100;

SELECT * FROM Users WHERE name LIKE 'A%';

SQL JOIN

JOIN combines rows from multiple tables. INNER JOIN returns matching rows. LEFT JOIN returns all left table rows. RIGHT JOIN returns all right table rows.

SQL Example
SELECT u.name, o.total
FROM Users u
INNER JOIN Orders o ON u.id = o.user_id;

SQL Aggregates & GROUP BY

Aggregate functions perform calculations: COUNT(), SUM(), AVG(), MAX(), MIN(). GROUP BY groups rows with same values. HAVING filters groups.

SQL Example
SELECT COUNT(*) FROM Users;

SELECT category, AVG(price) FROM Products GROUP BY category;
