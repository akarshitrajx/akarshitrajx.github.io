📌 1. HTML INTRODUCTION

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements (tags).

  • HTML stands for Hyper Text Markup Language
  • It is not a programming language — it's a markup language
  • It uses tags like <h1>, <p>, <div>
  • Every HTML page starts with <!DOCTYPE html>
🧪 TRY IT YOURSELF

📄 2. HTML BASIC

A simple HTML document structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>
  • <!DOCTYPE html> — declares HTML5
  • <html> — root element
  • <head> — metadata, title, styles
  • <body> — visible content
🧪 TRY IT YOURSELF

🧩 3. HTML ELEMENTS

An HTML element is everything from the start tag to the end tag:

<tagname>Content goes here...</tagname>
  • <h1>Heading</h1> — heading element
  • <p>Paragraph</p> — paragraph element
  • Empty elements (no closing tag): <br>, <hr>
  • Elements can be nested: <div><p>Text</p></div>

🏷️ 4. HTML ATTRIBUTES

Attributes provide additional information about elements:

<a href="https://example.com">Link</a>
<img src="photo.jpg" alt="My Photo" width="300" height="200">
<p class="intro" id="para1">This is a paragraph.</p>
<input type="text" placeholder="Enter name">
  • href — link destination
  • src — image source
  • alt — alternative text
  • width / height — size
  • class — CSS class
  • id — unique identifier
  • style — inline CSS
  • placeholder — hint text

🎨 11. HTML COLORS

Colors can be defined by:

  • Color names: red, blue, green, tomato, gold
  • HEX: #FF0000, #00FF00, #0000FF
  • RGB: rgb(255,0,0), rgb(0,255,0)
  • RGBA: rgba(255,0,0,0.5) (with transparency)
  • HSL: hsl(0,100%,50%)
💡 Tip: Use HEX for precise colors, RGB for digital design, HSL for intuitive adjustments.

📦 12. HTML CSS

Three ways to add CSS:

  • Inline: <h1 style="color:red;">
  • Internal: <style> inside <head>
  • External: <link rel="stylesheet" href="style.css">
<!-- External CSS -->
<link rel="stylesheet" href="styles.css">

<!-- Internal CSS -->
<style>
    body { background: #f0f0f0; }
    h1 { color: navy; }
    .highlight { background: yellow; }
</style>

📦 18. HTML BLOCK & INLINE

Block-level: Takes full width, starts new line

<div>, <h1>, <p>, <ul>, <table>, <form>

Inline: Takes only needed width, stays in line

<span>, <a>, <img>, <strong>, <em>, <input>

⚖️ 36. HTML vs XHTML

  • HTML: More flexible, case-insensitive, forgiving
  • XHTML: Stricter, XML-based, all tags must close, lowercase
  • XHTML requires <br /> instead of <br>
  • Modern websites use HTML5 (not XHTML)
