Python If-Else Complete Guide

If, Elif, Else, Nested If, Ternary Operator — conditions master karein!

🔄 If-Else Statements Kya Hain?

If-else se program decision le sakta hai. Agar kuch condition true hai toh ek kaam karo, nahi toh doosra kaam karo.

1️⃣ Basic If Statement

# Basic if age = 18 if age >= 18: print(<"Aap vote de sakte hain!")

2️⃣ If-Else Statement

# If-Else marks = 75 if marks >= 40: print(<"Pass! 🎉") else: print(<"Fail. Dobara try karein.")

3️⃣ If-Elif-Else (Multiple Conditions)

# If-Elif-Else marks = 85 if marks >= 90: print(<"Grade A+ 🏆") elif marks >= 80: print(<"Grade A 👍") elif marks >= 70: print(<"Grade B ✅") elif marks >= 60: print(<"Grade C") else: print(<"Grade D - Improvement needed")

4️⃣ Nested If (If ke andar If)

# Nested If age = 20 has_id = True if age >= 18: if has_id: print(<"Aap andar ja sakte hain!") else: print(<"ID card laiye!") else: print(<"Aap chhote hain.")

5️⃣ Ternary Operator (One-line If-Else)

# Ternary Operator age = 17 status = "Adult" if age >= 18 else "Minor" print(status) # Output: Minor # Another example marks = 85 result = "Pass" if marks >= 40 else "Fail" print(result) # Output: Pass

6️⃣ Logical Operators ke saath If

# and, or, not with if age = 17 marks = 85 if age >= 15 and marks >= 80: print(<"Scholarship mil sakti hai! 🎓") if age < 18 or marks > 90: print(<"Special category") if not (age < 13): print(<"Teenager ya adult hai")

7️⃣ Real Example: Calculator

# Simple calculator using if-elif a = 10 b = 5 operator = "+" if operator == "+": result = a + b elif operator == "-": result = a - b elif operator == "*": result = a * b elif operator == "/": if b != 0: result = a / b else: result = "Cannot divide by zero!" else: result = "Invalid operator" print(<f"Result: {result}")
💡 Indentation Important: Python mein if-else ke andar wala code 4 spaces indent hona chahiye. Indentation galat hui toh error aayega!
