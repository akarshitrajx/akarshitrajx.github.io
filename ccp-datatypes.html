🔢 C++ Data Types Complete Guide

C++ mein har variable ka ek fixed data type hota hai. Type batata hai ki variable mein kis type ka data store hoga aur kitni memory use hogi.

📊 Data Types Overview

TypeSizeUseExample
int4 bytesWhole numbers10, -5, 1000
float4 bytesDecimal numbers5.5, 3.14
double8 bytesBade decimals5.55555555
char1 byteSingle character'A', 'x', '@'
bool1 byteTrue/Falsetrue, false
stringVariableText"Hello"

1️⃣ int (Integer)

int age = 17;
int year = 2026;
int temperature = -5;
cout << age << endl;
cout << year << endl;

2️⃣ float (Floating Point)

float height = 5.8f;
float pi = 3.14159f;
float price = 99.99f;
cout << height << endl;
cout << pi << endl;

3️⃣ double (Double Precision)

double precise = 3.141592653589793;
double bigNumber = 123456789.123456789;
cout << precise << endl;

4️⃣ char (Character)

char grade = 'A';
char symbol = '@';
char digit = '7';
cout << grade << endl;
cout << symbol << endl;

5️⃣ bool (Boolean)

bool isPassed = true;
bool isRaining = false;
cout << isPassed << endl;   // 1
cout << isRaining << endl;  // 0

6️⃣ string (Text)

#include <string>
string name = "Akarshit Raj";
string city = "Sitapur";
cout << name << endl;
cout << city << endl;

🚀 Type Conversion (Casting)

int x = 10;
cout << (float)x << endl;   // 10.0

float y = 3.7;
cout << (int)y << endl;     // 3

int a = 65;
cout << (char)a << endl;    // A
⚠️ Dhyaan: Bade number ko chhote type mein convert karne se data loss ho sakta hai!
💡 Pro Tip: Precision chahiye toh double use karein. Memory bachani hai toh float use karein.
