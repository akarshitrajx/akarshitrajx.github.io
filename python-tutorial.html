Python Variables & Data Types

Variables are containers for storing data. Python has no command for declaring a variable - it is created when you first assign a value to it. Use type() to check the data type.

Python Example
x = 5           # int
y = 'Hello'     # str
z = 3.14        # float
is_valid = True # bool

print(type(x))
print(type(y))

Python If...Else

Python supports logical conditions from mathematics. Use if, elif, and else for decision making. Python uses indentation (whitespace) to define code blocks.

Python Example
a = 200
b = 33

if b > a:
  print('b is greater')
elif a == b:
  print('equal')
else:
  print('a is greater')

Python Loops

Python has for loops and while loops. For loops iterate over sequences. While loops run while a condition is true. The range() function generates number sequences.

Python Example
# For loop
for i in range(5):
  print(i)

# While loop
i = 0
while i < 5:
  print(i)
  i += 1

Python Functions

A function is a block of code that runs when called. Functions can take parameters and return values. Use the def keyword to define a function.

Python Example
def greet(name):
  return 'Hello, ' + name

print(greet('Alice'))

# Default parameter
def power(base, exp=2):
  return base ** exp

print(power(3))
print(power(2, 3))
