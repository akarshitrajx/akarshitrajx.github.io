📝 Python Strings Complete Guide

String Python mein text data store karne ka sabse common way hai. Single quotes ya double quotes ke andar likha jata hai. Strings immutable hain — matlab ek baar banne ke baad change nahi ho sakti!

1️⃣ String Kaise Banayein?

name = "Akarshit Raj"
city = 'Sitapur'
message = """Ye multi-line
string hai!"""
single_char = 'A'

print(name)
print(city)
print(message)
print(single_char)

2️⃣ String Indexing & Slicing

Har character ka apna index hota hai — 0 se shuru hota hai.

text = "Python"

print(text[0])      # P
print(text[1])      # y
print(text[-1])     # n (last character)
print(text[-2])     # o (second last)

# Slicing [start:end:step]
print(text[0:3])    # Pyt
print(text[:4])     # Pyth
print(text[2:])     # thon
print(text[::2])    # Pto (har doosra character)
print(text[::-1])   # nohtyP (reverse)

3️⃣ String Length & Check

text = "Hello World"

print(len(text))              # 11
print("World" in text)       # True
print("Python" not in text)  # True

4️⃣ String Methods (Important!)

text = "  Hello Python  "

print(text.upper())           # HELLO PYTHON
print(text.lower())           # hello python
print(text.strip())           # Hello Python (spaces hata dega)
print(text.replace("Python", "World"))
print(text.split())           # ['Hello', 'Python']

name = "akarshit raj"
print(name.capitalize())      # Akarshit raj
print(name.title())           # Akarshit Raj
print(name.startswith("akar")) # True
print(name.endswith("raj"))   # True
print(name.find("shit"))      # 4
print(name.count("a"))        # 3

# Join method
print("-".join(["a", "b", "c"]))  # a-b-c

5️⃣ String Formatting

3 tarike hain strings mein variables insert karne ke:

name = "Akarshit"
age = 17
marks = 85.5

# f-strings (Sabse Best aur Easy!)
print(f"Mera naam {name} hai, age {age} hai, aur marks {marks} hain.")

# format() method
print("Mera naam {} hai aur age {} hai.".format(name, age))

# % formatting (Purana tarika)
print("Mera naam %s hai aur age %d hai." % (name, age))
💡 Pro Tip: f-strings Python 3.6+ mein sabse fast aur readable hai. Hamesha f-strings use karein!

6️⃣ Escape Characters

Special characters string mein dikhane ke liye:

print("Line 1\\nLine 2")        # New line
print("Tab\\tSpace")            # Tab space
print("Ye \\"quotes\\" hain")     # Double quotes
print('Ye \\'quotes\\' hain')     # Single quotes
print("Path: C:\\\\Users\\\\Akarshit")  # Backslash

7️⃣ String Immutability

String ek baar banne ke baad change nahi ho sakti:

name = "Python"
# name[0] = "J"      # ❌ Error! String change nahi ho sakti

# Lekin nayi string bana sakte hain:
name = "Java"                    # ✅ Nayi string assign
new_name = "J" + name[1:]        # ✅ Nayi string bana sakte hain
print(new_name)

# Replace bhi nayi string deta hai, purani nahi badalta
text = "Hello World"
new_text = text.replace("World", "Python")
print(text)       # Hello World (purani same)
print(new_text)   # Hello Python (nayi string)
⚠️ Dhyaan: String immutable hai — har method nayi string return karta hai. Original string kabhi change nahi hoti!

8️⃣ String Loop

for char in "Python":
    print(char)

# Output:
# P
# y
# t
# h
# o
# n

9️⃣ Practical Examples

# Reverse a string
text = "Akarshit"
print(text[::-1])           # tihsrakA

# Palindrome check
word = "madam"
print(word == word[::-1])   # True

# Count vowels
text = "Hello World"
vowels = sum(1 for char in text.lower() if char in "aeiou")
print(f"Vowels: {vowels}")  # Vowels: 3

# Remove all spaces
sentence = "Hello World Python"
print(sentence.replace(" ", ""))  # HelloWorldPython

# Check if string is digit
num = "12345"
print(num.isdigit())        # True

# Check if alphabetic
name = "Akarshit"
print(name.isalpha())       # True
💡 Yaad Rakhein:
• Strings immutable hain — direct change nahi kar sakte
• f-strings sabse easy formatting method hai
• Slicing [start:end:step] bahut powerful hai
• Always use .strip() user input se pehle!
• String methods hamesha nayi string return karte hain
