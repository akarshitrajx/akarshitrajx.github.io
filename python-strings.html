📝 Python Strings Complete Guide

String text ya characters ka sequence hai. Python mein strings immutable hain — ek baar banane ke baad change nahi ho sakti. Lekin nayi string bana sakte hain!

1️⃣ String Kaise Banayein?

# Creating strings name = "Akarshit Raj" city = 'Sitapur' multiline = """This is a multiline string""" print(name) print(city) print(multiline)

2️⃣ String Indexing & Slicing

# Indexing text = "Python" print(text[0]) # P print(text[1]) # y print(text[-<1]) # n (last) print(text[-<2]) # o (second last) # Slicing print(text[0:<3]) # Pyt print(text[:<4]) # Pyth print(text[2:]) # thon print(text[::-<1]) # nohtyP (reverse)

3️⃣ String Methods

# Case methods text = "Hello World" print(text.upper()) # HELLO WORLD print(text.lower()) # hello world print(text.title()) # Hello World print(text.capitalize()) # Hello world # Search methods print(text.find("World")) # 6 print(text.replace("World", "Python")) # Hello Python print(<"World" in text) # True # Strip whitespace print(<" hello ".strip()) # "hello" print(<" hello ".lstrip()) # "hello " print(<" hello ".rstrip()) # " hello" # Split & Join words = "apple,banana,mango".split(",") print(words) # ['apple', 'banana', 'mango'] new_text = "-".join(words) print(new_text) # apple-banana-mango

4️⃣ String Formatting

# f-strings (Recommended) name = "Akarshit" age = 17 print(<f"My name is {name} and I am {age} years old.") # .format() print(<"My name is {} and I am {} years old.".format(name, age)) # % formatting print(<"My name is %s and I am %d years old." % (name, age)) # Expressions in f-strings print(<f"Next year I will be {age + 1}") print(<f"Name in uppercase: {name.upper()}")

5️⃣ String Checking Methods

# Check string content text = "Python123" print(text.isalpha()) # False (has numbers) print(text.isdigit()) # False print(text.isalnum()) # True (letters + numbers) print(text.isspace()) # False print(text.startswith("Py")) # True print(text.endswith("123")) # True

6️⃣ Escape Characters

# Escape sequences print(<"Hello\nWorld") # New line print(<"Hello\tWorld") # Tab print(<"It\'s a book") # Single quote print(<'He said "Hello"') # Double quote print(<"Path: C:\\Users\\Name") # Backslash
💡 Best Practice: Hamesha f-strings use karein formatting ke liye — ye sabse readable aur fast hai! Python 3.6+ mein available hai.
