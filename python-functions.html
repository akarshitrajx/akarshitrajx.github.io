Python Functions Complete Guide

Def, Parameters, Return, Default Args, *args, **kwargs, Lambda — sab functions!

🔧 Python Functions Kya Hain?

Function ek reusable code block hai jo specific kaam karta hai. Baar-baar use hone wala code ek function mein daal dein, aur jab chahiye tab call kar lein!

1️⃣ Function Kaise Banayein?

# Basic function def greet(): print(<"Hello, Welcome to Python!") # Function call greet() # Output: Hello, Welcome to Python! greet() # Output: Hello, Welcome to Python!

2️⃣ Function with Parameters

# Function with parameters def greet_user(name): print(<f"Hello, {name}! Kaise hain aap?") greet_user(<"Akarshit") # Hello, Akarshit! Kaise hain aap? greet_user(<"Rahul") # Hello, Rahul! Kaise hain aap?

3️⃣ Function with Return Value

# Function with return def add_numbers(a, b): result = a + b return result sum = add_numbers(<10, 20) print(sum) # Output: 30 # Direct use print(<add_numbers(<5, 3)) # Output: 8

4️⃣ Default Parameters

# Default parameter values def greet(name="Guest", message="Welcome"): print(<f"{message}, {name}!") greet() # Welcome, Guest! greet(<"Akarshit") # Welcome, Akarshit! greet(<"Rahul", "Hello") # Hello, Rahul!

5️⃣ Keyword Arguments

# Keyword arguments def student_info(name, age, school): print(<f"{name} is {age} years old and studies at {school}") student_info(age=17, school="JNV", name="Akarshit") # Output: Akarshit is 17 years old and studies at JNV

6️⃣ *args (Variable Arguments)

# *args - multiple arguments def total_marks(*marks): return sum(marks) result = total_marks(<85, 90, 78, 92) print(result) # Output: 345

7️⃣ **kwargs (Keyword Arguments)

# **kwargs - dictionary of arguments def print_details(**kwargs): for key, value in kwargs.items(): print(<f"{key}: {value}") print_details(name="Akarshit", age=17, city="Sitapur") # Output: # name: Akarshit # age: 17 # city: Sitapur

8️⃣ Lambda Functions (One-line)

# Lambda function square = lambda x: x ** 2 print(<square(<5)) # Output: 25 add = lambda a, b: a + b print(<add(<10, 20)) # Output: 30 # With map() numbers = [1, 2, 3, 4, 5] squares = list(<map(<lambda x: x**<2, numbers)) print(squares) # [1, 4, 9, 16, 25]

9️⃣ Real Example: Calculator Function

# Complete calculator def calculator(a, b, operation="+"): if operation == "+": return a + b elif operation == "-": return a - b elif operation == "*": return a * b elif operation == "/": return a / b if b != 0 else "Error: Division by zero" else: return "Invalid operation" print(<calculator(<10, 5, "+")) # 15 print(<calculator(<10, 5, "*")) # 50
💡 DRY Principle: Don't Repeat Yourself! Jo code baar-baar use hota hai, usse function mein daal dein. Code clean aur maintainable rahega!
