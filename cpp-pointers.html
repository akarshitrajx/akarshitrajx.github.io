C++ Pointers Complete Guide

What is a Pointer?

A pointer stores the memory address of a variable.

int age = 18;
int* ptr = &age;    // & = address of

cout << age;        // 18
cout << ptr;        // memory address
cout << *ptr;       // 18 (dereference)

Pointer Declaration

int* p1;      // pointer to int
double* p2;   // pointer to double
char* p3;     // pointer to char

Pointer Arithmetic

int arr[] = {10, 20, 30};
int* ptr = arr;

cout << *ptr;     // 10
ptr++;
cout << *ptr;     // 20
ptr++;
cout << *ptr;     // 30

Dynamic Memory

int* p = new int;     // allocate
*p = 25;
cout << *p;           // 25
delete p;             // deallocate

int* arr = new int[5];  // array
delete[] arr;           // deallocate array
