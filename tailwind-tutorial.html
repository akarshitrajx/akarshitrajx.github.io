Tailwind CSS Introduction

Tailwind CSS is a utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup. Unlike Bootstrap, Tailwind doesn't provide pre-built components - you build your own.

Tailwind Flexbox & Grid

Tailwind provides comprehensive flexbox and grid utilities. Use flex, flex-col, justify-center, items-center for flexbox. Use grid, grid-cols-3, gap-4 for grid layouts. Responsive prefixes like md: and lg: enable responsive designs.

Tailwind Forms & Components

Tailwind makes form styling easy with utility classes. Use focus:ring, focus:border-color for focus states. Create cards, buttons, alerts, and more using combinations of utility classes. The @apply directive helps extract component classes.
