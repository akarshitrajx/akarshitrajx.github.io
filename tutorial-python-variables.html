Python Variables Kya Hain?

Python 📅 June 6, 2026 ⏱ 8 min read

Variables programming ka sabse important aur basic concept hai. Bina variables ke koi bhi program complete nahi ho sakta. Aaj hum Python mein variables ko detail mein samjhenge — ye kya hain, kaise kaam karte hain, aur inhe kaise use karte hain.

🔷 Variable Kya Hai?

Simple words mein, variable ek container hai jismein aap data store kar sakte hain. Jaise aap ek dabbe mein kuch rakh ke uska naam likh dete hain, waise hi variable ek memory location ka naam hai jismein aap values store karte hain. Python mein variables dynamically typed hote hain, matlab aapko batane ki zarurat nahi ki ismein number hoga ya text — Python khud samajh jata hai!

Example ke taur par, agar aapko kisi ka naam ya age store karni hai, toh aap ek variable bana ke usmein value rakh sakte hain. Ye value baad mein change bhi ho sakti hai, isliye iska naam "variable" hai — matlab jo badal sakta hai.

📝 Variables Kaise Banayein?

Python mein variable banane ke liye koi special keyword nahi chahiye. Bas naam likhiye, equal sign lagaiye, aur value de dijiye. Itna simple!

# String variable
name = "Akarshit"

# Integer variable
age = 17

# Float variable
height = 5.8

# Boolean variable
is_student = True

print(name)
print(age)
print(height)
print(is_student)

🔢 Python Ke Data Types

Python mein alag-alag types ke variables hote hain. Har type ka apna use hota hai:

1. int (Integer): Ye whole numbers ke liye hota hai jaise 10, 25, 1000, -5. Jab aap counting, age, ya koi bhi number bina decimal ke store karna chahte hain, tab int use hota hai.

2. float (Floating Point): Ye decimal numbers ke liye hota hai jaise 5.8, 3.14, 99.99. Jab precise value chahiye ho jaise height, weight, ya percentage, tab float use karte hain.

3. str (String): Ye text ke liye hota hai. Single quotes ya double quotes ke andar likha jata hai. Naam, address, email — sab string mein store hote hain.

4. bool (Boolean): Ye sirf do values le sakta hai — True ya False. Conditions check karne mein, if-else statements mein ye bahut kaam aata hai.

5. list, tuple, dict: Ye advanced data types hain jo multiple values ek saath store karne ke liye use hote hain. Inke baare mein hum agli tutorial mein detail mein padhenge.

# Different data types
name = "Akarshit"           # str
age = 17                  # int
marks = 89.5              # float
passed = True              # bool

print(<type(name))       # <class 'str'>
print(<type(age))        # <class 'int'>
print(<type(marks))      # <class 'float'>
print(<type(passed))     # <class 'bool'>

✏️ Variable Ki Value Kaise Change Karein?

Variables ka sabse bada fayda ye hai ki inki value baad mein change ki ja sakti hai. Aap ek hi variable ko baar-baar alag-alag values assign kar sakte hain.

score = 50
print(<"Pehla score:", score)    # 50

score = 75
print(<"Naya score:", score)      # 75

score = score + 10
print(<"Final score:", score)    # 85

⚠️ Variable Naming Rules

Python mein variables ke naam rakhne ke kuch rules hain jo aapko follow karne hain:

• Variable ka naam letter (a-z, A-Z) ya underscore (_) se shuru hona chahiye. Number se nahi.

• Naam mein sirf letters, numbers, aur underscores hi allowed hain. Spaces nahi!

• Python case-sensitive hai, matlab Name aur name alag variables hain.

• Reserved keywords jaise if, for, class, True ko variable naam nahi bana sakte.

# Sahi naam
my_name = "Akarshit"
_age = 17
student1 = "Rahul"

# Galat naam — ye error dega!
# 1student = "Rahul"    # Number se start
# my-name = "Akarshit"  # Hyphen allowed nahi
# class = "Python"      # Reserved keyword
💡 Pro Tip: Hamesha meaningful naam use karein! x ya a ke bajaye student_name ya total_marks likhein. Isse code padhne mein aasani hoti hai aur baad mein samajhne mein bhi madad milti hai.

🎯 Multiple Variables Ek Saath Banana

Python mein aap ek hi line mein multiple variables assign kar sakte hain. Ye code chhota aur clean dikhta hai.

# Ek line mein multiple variables
a, b, c = 10, 20, 30
print(a, b, c)    # 10 20 30

# Same value multiple variables
x = y = z = 100
print(x, y, z)    # 100 100 100

🚀 Conclusion

Variables Python programming ka foundation hai. Inke bina koi bhi program complete nahi ho sakta. Aaj humne samjha ki variable kya hai, kaise banate hain, alag-alag data types kya hain, aur naming rules kya hain. Practice karein — jitna zyada code likhenge, utna accha samajh aayega!
