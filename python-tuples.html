📦 Python Tuples Complete Guide

Tuple list jaisa hi hai, lekin immutable — matlab ek baar banne ke baad change nahi ho sakta. Round brackets () mein likha jata hai. Fixed data store karne ke liye best hai!

1️⃣ Tuple Kaise Banayein?

fruits = ("Apple", "Banana", "Mango")
numbers = (1, 2, 3, 4, 5)
mixed = (1, "hello", 3.14, True)
single = ("Python",)    # Single item ke baad comma zaruri hai!
empty = ()

print(fruits)
print(numbers)
print(type(single))     # <class 'tuple'>

# Without comma, string ban jayegi
not_tuple = ("Python")
print(type(not_tuple))  # <class 'str'> — Ye tuple nahi hai!
⚠️ Dhyaan: Single item tuple mein comma (,) zaruri hai! Bina comma ke string ban jayegi.

2️⃣ Accessing Items (Indexing & Slicing)

fruits = ("Apple", "Banana", "Mango", "Orange", "Grapes")

print(fruits[0])        # Apple
print(fruits[2])        # Mango
print(fruits[-1])       # Grapes (last)
print(fruits[-2])       # Orange (second last)

# Slicing
print(fruits[1:4])      # ('Banana', 'Mango', 'Orange')
print(fruits[:3])       # ('Apple', 'Banana', 'Mango')
print(fruits[::2])      # ('Apple', 'Mango', 'Grapes')
print(fruits[::-1])     # ('Grapes', 'Orange', 'Mango', 'Banana', 'Apple')

3️⃣ Tuple Immutability

fruits = ("Apple", "Banana", "Mango")
# fruits[0] = "Orange"    # ❌ TypeError! Tuple change nahi ho sakta

# Lekin agar andar mutable item ho toh usko change kar sakte hain
t = (1, 2, [3, 4])
t[2][0] = 30            # ✅ List andar change ho sakti hai
print(t)                # (1, 2, [30, 4])

# Tuple ko list mein convert karke change kar sakte hain
fruits = ("Apple", "Banana", "Mango")
fruits_list = list(fruits)
fruits_list[0] = "Orange"
fruits = tuple(fruits_list)
print(fruits)           # ('Orange', 'Banana', 'Mango')

4️⃣ Tuple Methods

numbers = (1, 2, 3, 2, 4, 2, 5)

print(numbers.count(2))     # 3 — kitni baar 2 aaya
print(numbers.index(4))     # 4 — 4 kis index par hai
print(len(numbers))         # 7 — total items

5️⃣ Tuple Unpacking

# Basic unpacking
a, b, c = (10, 20, 30)
print(a, b, c)            # 10 20 30

# Asterisk unpacking
fruits = ("Apple", "Banana", "Mango", "Orange")
first, *rest = fruits
print(first)              # Apple
print(rest)               # ['Banana', 'Mango', 'Orange']

first, second, *remaining = fruits
print(first, second)      # Apple Banana
print(remaining)          # ['Mango', 'Orange']

# Ignore karna ho toh underscore use karein
a, _, c = (1, 2, 3)
print(a, c)               # 1 3

7️⃣ Tuple as Dictionary Key

Tuple immutable hone ki wajah se dictionary key ban sakta hai (list nahi ban sakti):

locations = {
    (0, 0): "Origin",
    (10, 20): "Point A",
    (30, 40): "Point B"
}

print(locations[(10, 20)])    # Point A
print(locations[(0, 0)])      # Origin

8️⃣ Tuple Comprehension (Generator Expression)

numbers = (1, 2, 3, 4, 5)

# Generator expression se tuple banana
squares = tuple(x**2 for x in numbers)
print(squares)              # (1, 4, 9, 16, 25)

evens = tuple(x for x in range(10) if x % 2 == 0)
print(evens)                # (0, 2, 4, 6, 8)

9️⃣ Nested Tuples

matrix = (
    (1, 2, 3),
    (4, 5, 6),
    (7, 8, 9)
)

print(matrix[0])            # (1, 2, 3)
print(matrix[1][2])         # 6

for row in matrix:
    for item in row:
        print(item, end=" ")
    print()
# Output: 1 2 3
#         4 5 6
#         7 8 9

🔟 Practical Examples

# Multiple values return karna
def min_max(numbers):
    return min(numbers), max(numbers)

result = min_max([3, 1, 4, 1, 5, 9])
print(result)               # (1, 9)

minimum, maximum = result
print(f"Min: {minimum}, Max: {maximum}")  # Min: 1, Max: 9

# Swap values (Python magic!)
a, b = 10, 20
a, b = b, a
print(f"a: {a}, b: {b}")    # a: 20, b: 10

# Coordinates store karna
point = (10, 20)
x, y = point
print(f"X: {x}, Y: {y}")    # X: 10, Y: 20
💡 Yaad Rakhein:
• Tuple immutable hai — data safety ke liye best
• List se fast aur memory efficient hai
• Dictionary key ban sakta hai (list nahi ban sakti)
• Single item mein comma zaruri hai
• Agar data change nahi hoga toh hamesha tuple prefer karein!
