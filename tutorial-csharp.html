C# Complete Tutorial

What is C#?

C# (C-Sharp) is a modern, object-oriented language developed by Microsoft. It runs on the .NET framework and is used for web, desktop, and game development (Unity).

Classes & Objects

class Student {
    public string Name { get; set; }
    public int Age { get; set; }
    
    public void Display() {
        Console.WriteLine($"{Name} - {Age}");
    }
}

Student s1 = new Student { Name = "Akarshit", Age = 18 };
s1.Display();
