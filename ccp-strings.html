C++ Strings Complete Guide

String Declaration

#include <string>
string name = "Akarshit";
string greeting = "Hello";

Common Methods

  • length() or size() — get length
  • append() — add text
  • substr(start, length) — extract substring
  • find() — find position
  • replace() — replace text
string s = "Hello World";
cout << s.length();        // 11
cout << s.substr(0, 5);  // Hello
s.append("!");
cout << s;                 // Hello World!

String Concatenation

string first = "Hello";
string second = "World";
string result = first + " " + second;
cout << result;  // Hello World

String to Number

string numStr = "123";
int num = stoi(numStr);        // 123
double d = stod("3.14");     // 3.14

int x = 100;
string s = to_string(x);     // "100"
