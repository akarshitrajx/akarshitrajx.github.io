Python Operators Complete Guide

Arithmetic, Comparison, Logical, Assignment, Identity, Membership, Bitwise — sab operators!

➕ Python Operators Kya Hain?

Operators special symbols hain jo variables aur values par operations perform karte hain. Python mein 7 types ke operators hain.

1️⃣ Arithmetic Operators

Basic math operations ke liye:

# Arithmetic Operators a = 10 b = 3 print(a + b) # Addition: 13 print(a - b) # Subtraction: 7 print(a * b) # Multiplication: 30 print(a / b) # Division: 3.333... print(a // b) # Floor Division: 3 print(a % b) # Modulo (Remainder): 1 print(a ** b) # Exponent (Power): 1000

2️⃣ Comparison Operators

Do values compare karne ke liye — result hamesha True ya False:

# Comparison Operators x = 10 y = 5 print(x == y) # Equal: False print(x != y) # Not Equal: True print(x > y) # Greater Than: True print(x < y) # Less Than: False print(x >= y) # Greater Than or Equal: True print(x <= y) # Less Than or Equal: False

3️⃣ Assignment Operators

Values assign karne ke liye — shortcut operators:

# Assignment Operators score = 50 # Basic assignment score += 10 # score = score + 10 → 60 score -= 5 # score = score - 5 → 55 score *= 2 # score = score * 2 → 110 score /= 2 # score = score / 2 → 55.0 print(score) # Output: 55.0

4️⃣ Logical Operators

Multiple conditions combine karne ke liye:

# Logical Operators age = 17 marks = 85 print(age > 15 and marks > 80) # and: True print(age > 20 or marks > 80) # or: True print(<not(age > 20)) # not: True

5️⃣ Identity Operators

Check karte hain ki do variables same object hain ya nahi:

# Identity Operators a = [1, 2, 3] b = a c = [1, 2, 3] print(a is b) # True (same object) print(a is c) # False (different object) print(a is not c) # True

6️⃣ Membership Operators

Check karte hain ki value sequence mein hai ya nahi:

# Membership Operators fruits = ["Apple", "Banana", "Mango"] print(<"Apple" in fruits) # True print(<"Grapes" in fruits) # False print(<"Grapes" not in fruits) # True name = "Akarshit" print(<"kar" in name) # True (substring check)

7️⃣ Bitwise Operators

Binary level par operations ke liye:

# Bitwise Operators a = 5 # Binary: 101 b = 3 # Binary: 011 print(a & b) # AND: 1 (001) print(a | b) # OR: 7 (111) print(a ^ b) # XOR: 6 (110) print(~a) # NOT: -6 print(a << 1) # Left Shift: 10 (1010) print(a >> 1) # Right Shift: 2 (10)
💡 Yaad Rakhein: // floor division hai (decimal hata deta hai), % remainder deta hai, aur ** power ke liye hai!
