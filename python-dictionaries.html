📖 Python Dictionaries Complete Guide

Dictionary key-value pairs store karne ke liye use hota hai. Curly brackets {} mein likha jata hai. Har key unique hoti hai!

1️⃣ Dictionary Kaise Banayein?

# Creating dictionaries student = { "name": "Akarshit Raj", "age": 17, "school": "JNV Khairabad", "city": "Sitapur" } empty_dict = {} dict() # Empty dict using function print(student) # {'name': 'Akarshit Raj', 'age': 17, 'school': 'JNV Khairabad', 'city': 'Sitapur'}

2️⃣ Accessing Values

# Access by key student = {"name": "Akarshit", "age": 17, "marks": 85} print(student["name"]) # Akarshit print(student["age"]) # 17 print(student.get("marks")) # 85 print(student.get("grade", "N/A")) # N/A (default if key not found)

3️⃣ Modifying Dictionaries

# Add/Update student = {"name": "Akarshit", "age": 17} student["school"] = "JNV Khairabad" # Add new key student["age"] = 18 # Update existing student.update({<"city": "Sitapur", "hobby": "Coding"}) # Multiple update # Remove student.pop("hobby") # Remove by key del student["city"] # Delete by key student.popitem() # Remove last item student.clear() # Empty the dict

4️⃣ Dictionary Methods

# Useful methods student = {"name": "Akarshit", "age": 17, "school": "JNV"} print(student.keys()) # dict_keys(['name', 'age', 'school']) print(student.values()) # dict_values(['Akarshit', 17, 'JNV']) print(student.items()) # dict_items([('name', 'Akarshit'), ...]) print(<"name" in student) # True print(<len(student)) # 3

5️⃣ Looping Through Dictionary

# Loop through keys for key in student: print(key) # Loop through values for value in student.values(): print(value) # Loop through key-value pairs for key, value in student.items(): print(<f"{key}: {value}")

6️⃣ Nested Dictionaries

# Nested dictionary classroom = { "student1": {"name": "Akarshit", "marks": 85}, "student2": {"name": "Rahul", "marks": 90}, "student3": {"name": "Priya", "marks": 78} } print(classroom["student1"][<"name"]) # Akarshit print(classroom["student2"][<"marks"]) # 90

7️⃣ Dictionary Comprehension

# Dictionary Comprehension squares = {x: x**<2 for x in range(<6)} print(squares) # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25} # From two lists keys = ["a", "b", "c"] values = [1, 2, 3] my_dict = {k: v for k, v in zip(keys, values)} print(my_dict) # {'a': 1, 'b': 2, 'c': 3}
💡 Dictionary vs List: List mein index se access karte hain (0, 1, 2...), Dictionary mein key se access karte hain ("name", "age"...). Jab aapko labeled data chahiye, tab dictionary use karein!
