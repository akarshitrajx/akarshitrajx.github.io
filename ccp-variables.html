📝 C++ Variables Complete Guide

Variables C++ programming ka sabse important concept hai. C++ mein har variable ka ek fixed type hota hai — isliye isse statically typed language kehte hain.

🔷 Variable Kya Hai?

Variable ek memory location ka naam hai jismein aap data store karte hain. C++ mein variable banane se pehle uska type batana zaruri hai.

📝 Variables Kaise Banayein?

#include <iostream>
using namespace std;

int main() {
    int age = 17;
    float height = 5.8;
    char grade = 'A';
    string name = "Akarshit";
    
    cout << "Name: " << name << endl;
    cout << "Age: " << age << endl;
    cout << "Height: " << height << endl;
    cout << "Grade: " << grade << endl;
    
    return 0;
}

🔢 C++ Ke Data Types

int number = 100;           // Whole numbers
float decimal = 5.5;      // Decimal numbers (4 bytes)
double bigDecimal = 5.555555; // Bade decimal (8 bytes)
char letter = 'A';          // Single character
bool isTrue = true;         // True ya False
string text = "Hello";       // Text string

✏️ Variable Ki Value Kaise Change Karein?

int score = 50;
cout << "Pehla score: " << score << endl;

score = 75;
cout << "Naya score: " << score << endl;

score = score + 10;
cout << "Final score: " << score << endl;

⚠️ Variable Naming Rules

  • Naam letter (a-z, A-Z) ya underscore (_) se shuru hona chahiye.
  • Number se shuru nahi ho sakta.
  • Spaces nahi allowed — use underscore.
  • Case-sensitive hai — Age aur age alag hain.
  • Reserved keywords (int, float, return, etc.) use nahi kar sakte.
int my_age = 17;      // ✅ Sahi
int _score = 85;      // ✅ Sahi
int student1 = 1;     // ✅ Sahi
// int 1student = 1;  // ❌ Galat — number se shuru
// int my age = 17;   // ❌ Galat — space hai
💡 Pro Tip: C++ mein variable declare karte waqt hamesha type likhein! int age = 17; — type + name + value.

🎯 Multiple Variables Ek Saath

int a = 10, b = 20, c = 30;
cout << a << " " << b << " " << c << endl;

int x, y, z;
x = y = z = 100;
cout << x << " " << y << " " << z << endl;
