TypeScript Introduction

TypeScript is a typed superset of JavaScript that compiles to plain JavaScript. It adds optional static typing, classes, and interfaces to JavaScript. TypeScript is developed and maintained by Microsoft. It helps catch errors early and improves code quality.

TypeScript Example
// TypeScript with types
let name: string = "Alice";
let age: number = 25;
let isActive: boolean = true;

// Function with types
function greet(name: string): string {
  return "Hello, " + name;
}

console.log(greet("Bob"));

TypeScript Interfaces & Types

Interfaces define the structure of an object. They are a powerful way to define contracts within your code. Type aliases create a new name for a type. Both help with type checking and code documentation.

TypeScript Example
// Interface
interface Person {
  name: string;
  age: number;
  email?: string;  // optional
}

// Type alias
type ID = string | number;

// Using interface
const user: Person = {
  name: "Alice",
  age: 25
};

// Function with interface
function printPerson(p: Person): void {
  console.log(p.name, p.age);
}

TypeScript Classes & Generics

TypeScript supports classes with inheritance, access modifiers (public, private, protected), and more. Generics provide a way to create reusable components that work with multiple types rather than a single one.

TypeScript Example
// Class with types
class Animal {
  private name: string;
  
  constructor(name: string) {
    this.name = name;
  }
  
  move(distance: number): void {
    console.log(`${this.name} moved ${distance}m`);
  }
}

// Generic function
function identity(arg: T): T {
  return arg;
}

let output = identity("myString");
