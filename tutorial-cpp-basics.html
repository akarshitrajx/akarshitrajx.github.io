C++ Basics - Getting Started

C++ 📅 June 5, 2026 ⏱ 10 min read

C++ ek powerful programming language hai jo Bjarne Stroustrup ne 1985 mein banayi thi. Ye C language ka extended version hai jismein Object-Oriented Programming (OOP) features add kiye gaye hain. Aaj C++ games, operating systems, browsers, aur high-performance applications banane mein use hoti hai. Agar aapko programming seriously seekhni hai, toh C++ se behtar start koi nahi!

🔷 C++ Kyun Seekhein?

C++ seekhne ke bahut saare reasons hain. Sabse pehla reason ye hai ki C++ fast hai. Ye directly machine code mein compile hoti hai, isliye ye Python ya Java se zyada tez chalti hai. Agar aapko game development ya system programming mein interest hai, toh C++ must hai.

Dusra reason ye hai ki C++ OOP concepts classes, objects, inheritance, polymorphism — sab sikhati hai. Ye concepts aapko Java, C#, Python ki OOP samajhne mein bhi madad karte hain. Ek baar C++ mein OOP clear ho jaye, baaki languages aasani se aa jati hain.

Teesra reason — C++ memory management sikhati hai. Aap khud control karte hain ki memory kaise allocate aur free hogi. Ye deep understanding baad mein debugging aur optimization mein kaam aati hai.

🛠️ Pehla C++ Program

Har programmer ki journey "Hello, World!" se shuru hoti hai. C++ mein ye program likhne ke liye aapko kuch basic cheezein samajhni hongi.

// Pehla C++ Program
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Line-by-line samajhte hain:

• #include <iostream> — Ye header file input-output operations ke liye hoti hai. cout aur cin isi se aate hain.

• using namespace std; — Ye line standard library ke functions ko bina std:: likhe use karne deti hai. Code chhota dikhata hai.

• int main() — Ye program ka entry point hai. Har C++ program main() function se shuru hota hai.

• cout — Console output ke liye. Screen pe print karne ke liye use hota hai.

• return 0; — Ye batata hai ki program successfully complete hua.

📦 Data Types in C++

C++ mein har variable ka ek type hota hai. Type batata hai ki variable mein kis type ka data store hoga aur kitni memory use hogi.

Basic Data Types:

• int — Integer numbers ke liye (4 bytes). Example: 10, -5, 1000.

• float — Decimal numbers ke liye (4 bytes). Example: 5.5, 3.14.

• double — Bade decimal numbers ke liye (8 bytes). Float se zyada precise.

• char — Single character ke liye (1 byte). Example: 'A', 'x', '@'.

• bool — True ya False value ke liye (1 byte).

• string — Text ke liye. Multiple characters ek saath.

#include <iostream>
#include <string>
using namespace std;

int main() {
    int age = 17;
    float height = 5.8;
    char grade = 'A';
    bool isPassed = true;
    string name = "Akarshit";

    cout << "Name: " << name << endl;
    cout << "Age: " << age << endl;
    cout << "Height: " << height << endl;
    cout << "Grade: " << grade << endl;
    cout << "Passed: " << isPassed << endl;

    return 0;
}

✏️ Variables aur Constants

C++ mein variable banane ke liye aapko pehle type batana padta hai, fir naam, fir value. Ye C++ ko statically typed language banata hai.

int marks = 85;           // Integer variable
float percentage = 85.5;  // Float variable
char section = 'B';      // Character variable

// Constant — value change nahi ho sakti
const float PI = 3.14159;
// PI = 3.14;    // Ye error dega!

const keyword se aap constant bana sakte hain jiski value ek baar set hone ke baad change nahi hoti. Ye security aur reliability ke liye use hota hai.

🧮 Operators in C++

Operators variables par operations perform karne ke liye hote hain. C++ mein alag-alag types ke operators hain:

Arithmetic Operators: + (addition), - (subtraction), * (multiplication), / (division), % (modulo/remainder).

Comparison Operators: == (equal), != (not equal), > (greater than), < (less than), >=, <=.

Logical Operators: && (AND), || (OR), ! (NOT).

int a = 10, b = 3;

cout << "Sum: " << (a + b) << endl;        // 13
cout << "Difference: " << (a - b) << endl; // 7
cout << "Product: " << (a * b) << endl;    // 30
cout << "Quotient: " << (a / b) << endl;   // 3
cout << "Remainder: " << (a % b) << endl;  // 1
💡 Pro Tip: cin se user input le sakte hain! cin >> variable; likhne se program user se value maangta hai. Isse interactive programs bana sakte hain.

🎯 User Input Lena

Static values ke alawa, aap user se real-time input bhi le sakte hain. Isse aapke programs dynamic aur useful hote hain.

#include <iostream>
using namespace std;

int main() {
    string name;
    int age;

    cout << "Apna naam likhein: ";
    cin >> name;

    cout << "Apni age likhein: ";
    cin >> age;

    cout << "Hello, " << name << "! Aap " << age << " saal ke hain." << endl;

    return 0;
}
