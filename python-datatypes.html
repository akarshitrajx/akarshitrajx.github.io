Python Data Types Complete Guide

int, float, str, bool, list, tuple, dict, set — sab kuch detail mein!

🔢 Python Data Types Kya Hain?

Data type batata hai ki ek variable mein kis type ka data stored hai. Python mein har value ka ek data type hota hai. Ye automatically detect ho jata hai — aapko explicitly declare karne ki zarurat nahi!

1️⃣ int (Integer)

Whole numbers ke liye use hota hai — positive, negative, ya zero.

# Integer examples age = 17 year = 2026 temperature = -5 print(age) # Output: 17 print(year) # Output: 2026 print(temperature) # Output: -5

2️⃣ float (Floating Point)

Decimal numbers ke liye — jinmein point ke baad digits hain.

# Float examples height = 5.8 pi = 3.14159 price = 99.99 print(height) # Output: 5.8 print(pi) # Output: 3.14159

3️⃣ str (String)

Text ya characters ke liye. Single quotes ya double quotes mein likha jata hai.

# String examples name = "Akarshit Raj" city = 'Sitapur' message = "Hello, World! 🌍" print(name) # Output: Akarshit Raj print(city) # Output: Sitapur

4️⃣ bool (Boolean)

Sirf do values — True ya False. Conditions aur comparisons mein use hota hai.

# Boolean examples is_student = True is_raining = False passed = 85 > 40 # True print(is_student) # Output: True print(is_raining) # Output: False

5️⃣ list (List)

Multiple values ek saath store karne ke liye. Square brackets [] mein likha jata hai.

# List examples marks = [85, 90, 78, 92] fruits = ["Apple", "Banana", "Mango"] mixed = [1, "hello", 3.14, True] print(marks) # Output: [85, 90, 78, 92] print(fruits) # Output: ['Apple', 'Banana', 'Mango']

6️⃣ tuple (Tuple)

List jaisa hi hai, lekin immutable — ek baar banane ke baad change nahi ho sakta. Round brackets () mein likha jata hai.

# Tuple examples coordinates = (10, 20) colors = ("Red", "Green", "Blue") print(coordinates) # Output: (10, 20) print(colors[0]) # Output: Red

7️⃣ dict (Dictionary)

Key-value pairs store karne ke liye. Curly brackets {} mein likha jata hai.

# Dictionary examples student = { "name": "Akarshit", "age": 17, "school": "JNV Khairabad", "marks": [85, 90, 78] } print(student["name"]) # Output: Akarshit print(student["age"]) # Output: 17

8️⃣ set (Set)

Unique values store karne ke liye. Duplicate values automatically remove ho jati hain.

# Set examples numbers = {1, 2, 3, 3, 3} # Duplicates removed print(numbers) # Output: {1, 2, 3} fruits = {"Apple", "Banana", "Apple"} print(fruits) # Output: {'Apple', 'Banana'}

🎯 type() Function se Data Type Check Karein

name = "Akarshit" age = 17 height = 5.8 is_student = True print(<type(name)) # <class 'str'> print(<type(age)) # <class 'int'> print(<type(height)) # <class 'float'> print(<type(is_student)) # <class 'bool'>
💡 Pro Tip: Python dynamically typed hai — matlab aapko pehle se type batane ki zarurat nahi. Python khud samajh jata hai! Lekin type() function se check karna acchi practice hai.

🚀 Type Conversion (Casting)

Ek type ko doosre type mein convert kar sakte hain:

# int to float x = 10 print(<float(x)) # Output: 10.0 # float to int y = 3.7 print(<int(y)) # Output: 3 (decimal hata deta hai) # int to string z = 100 print(<str(z)) # Output: "100" # string to int s = "50" print(<int(s)) # Output: 50
⚠️ Dhyaan: "hello" ko int mein convert nahi kar sakte! Isse error aayegi. Sirf valid strings hi convert hongi.
