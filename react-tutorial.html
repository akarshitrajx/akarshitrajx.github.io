React Introduction

React is a JavaScript library for building user interfaces. It is maintained by Facebook and a community of developers. React allows you to build single-page applications with reusable components. It uses a virtual DOM for efficient updates.

React Example
import React from 'react';

function Hello() {
  return 
Hello, World!
;
}

export default Hello;

React Components & Props

Components are independent and reusable pieces of code. They are like JavaScript functions that return HTML elements. Props are arguments passed into React components. Props are passed via HTML attributes.

React Example
// Function Component with Props
function Welcome(props) {
  return 
Hello, {props.name}
;
}

// Class Component
class Welcome extends React.Component {
  render() {
    return 
Hello, {this.props.name}
;
  }
}

React Hooks

Hooks allow you to use state and other React features in functional components. useState manages state, useEffect handles side effects, useContext accesses context, and useRef creates mutable references.

React Example
import { useState, useEffect } from 'react';

function Timer() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    const timer = setInterval(() => {
      setCount(c => c + 1);
    }, 1000);
    return () => clearInterval(timer);
  }, []);

  return 
Count: {count}
;
}
