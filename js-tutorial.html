JS Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, or const. The let and const keywords were added in ES6 (2015). Use const when you declare a constant value.

JavaScript Example
let name = "John";
const age = 30;
var city = "New York";

console.log(name); // John
console.log(age);  // 30

JS Functions

A JavaScript function is a block of code designed to perform a particular task. A function is executed when something invokes it (calls it). Functions can take parameters and return values.

JavaScript Example
function greet(name) {
  return "Hello, " + name + "!";
}

console.log(greet("Alice")); // Hello, Alice!
console.log(greet("Bob"));   // Hello, Bob!

JS Arrays

Arrays are used to store multiple values in a single variable. Array items are separated by commas. You can access array elements by referring to the index number. Array indexes start at 0.

JavaScript Example
const fruits = ["Apple", "Banana", "Orange"];
console.log(fruits[0]); // Apple
console.log(fruits.length); // 3

fruits.push("Mango");
console.log(fruits); // ["Apple", "Banana", "Orange", "Mango"]

JS DOM Manipulation

The DOM (Document Object Model) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. JavaScript can change HTML content, attributes, styles, and react to events.

JavaScript Example
// Change HTML content
document.getElementById("demo").innerHTML = "New text";

// Change HTML style
document.getElementById("demo").style.color = "red";

// Add event listener
document.getElementById("btn").addEventListener("click", function() {
  alert("Button clicked!");
});
