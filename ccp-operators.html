C++ Operators Complete Guide

Arithmetic Operators

int a = 10, b = 3;
cout << a + b;  // 13
cout << a - b;  // 7
cout << a * b;  // 30
cout << a / b;  // 3
cout << a % b;  // 1

Comparison Operators

  • == Equal to
  • != Not equal
  • > Greater than
  • < Less than
  • >= Greater than or equal
  • <= Less than or equal

Logical Operators

bool a = true, b = false;
cout << (a && b);  // false (AND)
cout << (a || b);  // true  (OR)
cout << (!a);      // false (NOT)

Increment / Decrement

int x = 5;
cout << ++x;  // 6 (pre-increment)
cout << x++;  // 6, then x becomes 7
cout << --x;  // 6 (pre-decrement)
